# SC-200 / AI-Cyber Portfolio

Software Engineering graduate (Honours, Cybersecurity extension) from Taylor's University, dual degree with UWE Bristol. Kuwait-based. Hands-on experience in PAM/IAM through internships at International Turnkey Systems (ITS) and Agates Group Co. Currently working as an IT Systems Operator at KNET (Kuwait's national payment network), while preparing for the Microsoft SC-200 certification.

## What I'm targeting
- SOC Analyst
- GRC / Compliance Analyst
- IT Operations roles with a security growth path

Focused on Kuwait's banking, oil and gas, and consulting sectors.

## What this repo proves
This is a working log, not a highlight reel. It tracks daily hands-on practice across:
- **Network fundamentals** — diagnostics, diagramming, traffic flow
- **SOC concepts** — triage, escalation, alert types, monitoring
- **IAM / PAM** — authentication vs authorization, least privilege, identity providers
- **GRC** — ISO 27001, NIST CSF, Kuwait regulatory frameworks (CBK, CITRA, PDP 2021)
- **M365 / Purview / Sentinel** — KQL queries, log analysis (added as covered)
- **AI-assisted security tooling** — a companion project (`ai-cyber-triage-agent`) exploring AI-assisted triage and reporting, with a human-review policy baked in from day one
