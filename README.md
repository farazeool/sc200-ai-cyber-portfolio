# SC-200 + AI Cyber Portfolio

Evidence-first portfolio built against the **SC-200 + AI Cyber 35-Day Exact Action Manual v2.0**.

## Environment

- M1 MacBook Pro, 16 GB RAM
- Windows 11 ARM in Parallels for Windows-only labs
- Wireshark and Cisco Packet Tracer
- Track A: authorised Microsoft tenant, or Track B: synthetic/no-tenant KQL
- Separate AI repository: `ai-cyber-triage-agent`

## Operating rules

1. Store actual proof here, not decorative notes.
2. Keep observed facts, assumptions, confidence, and recommendations separate.
3. Use `tested` only for work actually run in the named environment.
4. Use `synthetic-manual-validation` or `draft-human-reviewed` otherwise.
5. Do not claim a day complete until its pass/fail gate is met.

## Portfolio areas

- `network/` — Packet Tracer, Wireshark, DNS/IP/ports, firewall and DMZ evidence
- `soc/` — tickets, incident reports, KQL, Sentinel/Defender work
- `grc-kuwait/` — risk, controls, audit, Kuwait regulatory context
- `iam-pam/` — JML, privileged access, vendor access and access reviews
- `m365-purview/` — Microsoft platform maps, Purview and M365 evidence
- `job-search/` — CVs, applications and recruiter material
- `interview/` — interview banks and mock interview evidence
- `notes/` — gate scorecards, recall and limited support notes
- `screenshots/` — only screenshots that prove work
